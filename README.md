# 4.atzime
Ši aplikācija demonstrē 15 dažadas python funckijas

Built-in Funckijas
1.bin() - converts an integer to a binary string
3.round() - rounds a number to a specified number of decimal places
4.sum() - returns the sum of all items in an iterable
5.complex() - converts a number or a string to a complex number
8.float() - converts a number or a string to a floating-point number

Random Funckijas
6.random() - returns a random float number between 0.0 and 1.0
7.randomint() - returns a random integer between a specified range
9.rand.uniform() - returns a random float number between a specified range
10.rand.choice() - returns a random element from a non-empty sequence

Matemātikās Funckijas
2.log10() - returns the base-10 logarithm of a number
11.fact() - returns the factorial of a number
12.sqrt() - returns the square root of a number
13.pow() - returns the value of a number raised to the power of another number
14.cbrt() - returns the cube root of a number
15.log() - returns the natural logarithm of a number
