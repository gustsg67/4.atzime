# 4.atzime
Ši aplikācija demonstrē 15 dažadas python funckijas

Built-in Funckijas
1.bin() - pārvērš veselu skaitli binārajā virknes formā
3.round() - noapaļo skaitli uz norādīto decimāldaļu skaitu
4.sum() - atgriež visu elementu summu iterējamā objektā
5.complex() - pārvērš skaitli vai virkni par kompleksu skaitli
8.float() - pārvērš skaitli vai virkni par "floating" punkta skaitli

Random Funckijas
6.random() - atgriež nejaušu "floating" skaitli no 0.0 līdz 1.0
7.randomint() - atgriež nejaušu skaitli starp norādīto diapazonu un 1,0
9.rand.uniform() - atgriež nejaušu "floating" skaitli norādītajā diapazonā
10.rand.choice() - atgriež nejaušu elementu no neaizpildītas secības

Matemātikās Funckijas
2.log10() - atgriež skaitļa desmitniekšķirnes logaritmu
11.fact() - atgriež skaitļa faktoriāli
12.sqrt() - atgriež skaitļa kvadrātsakni
13.pow() - atgriež skaitļa vērtību, paceltu citas skaitļa pakāpē
14.cbrt() - atgriež skaitļa kubsakni
15.log() - atgriež skaitļa dabisko logaritmu
